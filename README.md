# San vs Moon

Game project testing the possibility of hosting `websocket` on VPS.

[![Screen](https://github.com/bbrojson/galvanize/blob/trunk/screenshot.png)](https://galvanize-ecru.vercel.app/)

## Overview

This project is a simple one-button game where user can choose between **Sun** or **Moon**.
This project is about exploring the feasibility and performance of hosting WebSocket server on VPS.

## Stack (needs to be updated)

- VPS, debian
- nginx
- pm2
- nodejs
- Frontend: nextjs (Vercel)
